Project description

ImageJ is an open source Java based image processing program extensively used in life sciences. It is designed with an open architecture that allows to create any user written Plugins. Our project aims to redesign the existing Trainable Weka Segmentation (TWS) of ImageJ. The idea is to make it generic and user friendly so that we can include the most significant state of the art like filters and machine learning frameworks like active learning, multi-instance learning designed by third party in a robust manner.
