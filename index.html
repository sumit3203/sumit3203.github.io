Project description

ImageJ is an open source Java based image processing program extensively used in life sciences. It is designed with an open architecture that allows to create any user written Plugins. Our project aims to redesign the existing Trainable Weka Segmentation (TWS) of ImageJ. The idea is to make it generic and user friendly so that we can include the most significant state of the art like filters and machine learning frameworks like active learning, multi-instance learning designed by third party in a robust manner.

Session Screen

Session is a provision for storing project data in the session file. It save the state of the system. First image illustrates the window for Session Screen. Session screen will load the existing(Previous) session of the project. It provide you the option to load the filter, Feature and learning Data. The "Choose Session File" button is used for loading session file from the local file system. The whole meta-data of the project will be saved in the same directory as selected by the user. The session file must be in JSON format
